Validation Rules:

Title: Required, 3-100 characters
-Cook Time: Must be valid number, > 0, ≤ 1440 minutes (24 hours)
-Servings: Required, valid number, > 0, ≤ 100
-Ingredients: At least one required
-Instructions: At least one step required

Error Display Features:
-Inline error messages below each field
-Red error borders on fields with validation failures
-Error messages clear when user starts typing
-General error banner at top for API failures
-Field errors reset when form is reset

User Experience:
-Errors only show on submit attempt (not while typing)
-Validation feedback is immediate and specific
-Users can correct errors field-by-field
